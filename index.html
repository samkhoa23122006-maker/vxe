Backend JavaScript libraries that power the system

Database layer with localStorage abstraction. Handles all CRUD operations for users, routes, buses, schedules, tickets, and transactions.

JavaScript Library

auth.js

Authentication manager. Handles login, registration, password recovery, session management, and role-based access control.

JavaScript Library

👥 Public Pages

Accessible to all visitors

index-home.html

Landing page with hero section, feature showcase, company statistics, and calls-to-action. The entry point to the system.

Open Page → Home Page

login.html

Authentication hub with three tabs: Login, Register (new account), and Forgot Password recovery flow. Password visibility toggle included.

Open Page → Auth Page

about.html

Company information page with vision, mission, team details, and contact information. Shows why customers should choose our service.

Open Page → Info Page

👤 Customer Pages

Available after login (minimum privilege: customer)

dashboard.html

Customer home page showing personalized greeting, quick action cards, and recent bookings. Main hub after login.

Login First → Dashboard

booking.html

Complete ticket booking system: route search → schedule selection → seat map → passenger info → payment → confirmation with QR code. NOW WITH: Advanced filters, sorting, promotions, real-time tracking!

Login First → Booking System

tracking.html

NEW! Real-time journey tracking with GPS map, driver contact, journey timeline, current speed, ETA, and booking history. Track your bus in real-time!

Login First → Journey Tracking

my-tickets.html

View all booked tickets with filtering by status (paid, unpaid, used, cancelled). Shows route, date, seat, price, and QR code.

Login First → Ticket List

ticket-detail.html

Detailed ticket view with QR code, route information, schedule details, passenger info, and action buttons for printing, downloading, sharing, and paying.

Login First → Ticket Detail

profile.html

User account management page. View profile information (name, email, phone) and change password with validation.

Login First → Account Settings

⚙️ Admin Pages

Available only to admin users

admin-dashboard.html

Admin home page showing key metrics (revenue, tickets sold, schedules, customers) and navigation menu to all management modules.

Login as Admin → Admin Home

admin-routes.html

Route management CRUD interface. Add new routes, view all routes with details, and delete routes. Essential for setting up travel options.

Login as Admin → Route Management

admin-buses.html

Bus inventory management. Add buses with type (VIP/Standard) and capacity, view fleet, and remove vehicles from service.

Login as Admin → Bus Management

admin-schedules.html

Schedule creation and management. Assign buses and drivers to routes with specific dates and times. Critical for operations.

Login as Admin → Schedule Management

admin-reports.html

Revenue analytics and reporting. View revenue stats, ticket sales, transaction history with date filtering. Export reports to PDF format.

Login as Admin → Reports & Analytics
